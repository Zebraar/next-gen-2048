# Next-Gen 2048 Cyberpunk Custom

Стильная и современная реализация классической игры 2048 в неоновом киберпанк-стиле с плавной анимацией движения плиток и глубокой кастомизацией игрового процесса.

## 🚀 Особенности проекта

- **Плавные анимации:** Плитки перемещаются и сливаются с использованием CSS-переходов и матричных трансформаций, обеспечивая максимальный визуальный отклик.
- **Глубокая кастомизация:** Прямо в интерфейсе игры можно настроить:
  - Размер игрового поля (от 3х3 до 8x8).
  - Шанс появления плитки «2» (от 10% до 100%).
  - Целевую максимальную плитку для победы (от 64 до 16384).
  - Цветовую гамму интерфейса (динамическое изменение оттенка HUE).
- **Сохранение прогресса:** Лучший результат (Best Score) сохраняется в `localStorage` браузера индивидуально для каждого размера сетки.
- **Адаптивный дизайн:** Игра корректно отображается как на настольных мониторах, так и на мобильных устройствах, автоматически пересчитывая размеры элементов при изменении экрана.

## 🛠 Технологический стек

- **HTML5** — семантическая структура игрового поля и панели настроек.
- **CSS3** — кастомные CSS-переменные (Variables), размытие фона (`backdrop-filter`), градиенты и анимации (`@keyframes`).
- **JavaScript (ES6+)** — объектно-ориентированная логика игры (классы), обработка событий клавиатуры и динамический расчет позиций элементов.

## 🎮 Как запустить игру локально

1. Клонируйте репозиторий или скачайте архив с файлами.
2. Убедитесь, что все файлы находятся в одной папке согласно архитектуре:
```text
next-gen-2048/
   ├── index.html
   ├── style.css
   └── game.js
```
3. Откройте файл index.html в своём браузере
4. Наслаждайтесь игрой! ❤️
